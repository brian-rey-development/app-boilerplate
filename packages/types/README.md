# @packages/types

## Purpose

Pure TypeScript entity types and shared primitives. Zero runtime dependencies.

## Exports

- `User`
- `Organization`
- `OrgMember`
- `OrgInvite`
- `AuthContext`
- `OrgContext`
- `AppError`
- `Result<T, E>`
- `Paginated<T>`
- `success`, `failure` helpers

## Dependencies

- None (zero runtime dependencies)

## Commands

- `pnpm typecheck` -- TypeScript type checking
- `pnpm lint` -- TypeScript type checking (same as typecheck)
